## ⚡ Featured Projects: System Internals

*These projects demonstrate my understanding of how distributed systems work under the hood.*

### 🗄️ [MySQL Replication & Sharding Demo](https://github.com/kumarsantosh3914/mysql-replication-sharding-demo)
* **Concept:** Database Scaling
* **Tech:** Docker, MySQL, ProxySQL (implied/custom logic)
* **Highlight:** A hands-on implementation of **horizontal scaling** concepts. Demonstrates **Master-Slave replication** for read/write splitting and **Database Sharding** logic to distribute data load across multiple nodes.

### ⚖️ [Custom Load Balancer](https://github.com/kumarsantosh3914/basic-load-balancer)
* **Concept:** Networking & Concurrency
* **Tech:** Go (Golang)
* **Highlight:** Built a functional Layer 7 Load Balancer from scratch. Implements **Round-Robin** and **Least-Connection** algorithms, handling concurrent request routing to healthy backend servers.

### 🛡️ [Circuit Breaker Implementation](https://github.com/kumarsantosh3914/CircuitBreakers)
* **Concept:** Fault Tolerance
* **Highlight:** Custom implementation of the **Circuit Breaker pattern** to prevent cascading failures in microservices. Handles state transitions (Closed -> Open -> Half-Open) to ensure system resilience.
